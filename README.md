# HealthyPal
HealthyPal is a diet, social media application.Users can check the calories of their meal and share it with others who are dieting as well.
We build a community of diet,to encourage and alert each other.

# 1.Register 
User have to create a account with unique username and email.



Afterthat, user have to input some personal body information into the application for TDEE calculation and adjust how much calories do a user needs based on their goal 

# 2.login
To login in to our application,you have to input your email and password only. 


# 3.home page 
After you finished the information form, you will reach our home page.On your left-hand side , it is a navigation bar. The first button'home' is for going back to the home page, button'
